2-Player Asteroids
--
Connection instructions
--
-For host type 'y' in console
-For client type 'n' in console
 - Then enter the ipv4 of the machine you ran the host on (if running two instances on the same machine use 127.0.0.1)

Gameplay instructions
--
W - Move forward
A & D - Rotate left and right
Spacebar - Shoot

You control the white ship 
Red ship is player 2 for each instance

Game rules
--
First player to 100 points wins
